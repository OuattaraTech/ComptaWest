Captures des vrais documents générés par ApeX
==============================================

Dépose ici les captures (PNG ou JPG) des documents que tu veux
voir défiler dans le dôme rotatif de la landing page.

Fichiers attendus (mêmes noms exactement) :

  facture.png         — capture de la facture certifiée FNE
  devis.png           — capture du devis / proforma
  bulletin.png        — capture du bulletin de paie CNPS
  bilan.png           — capture du bilan SYSCOHADA
  amortissement.png   — capture du plan d'amortissement
  grandlivre.png      — capture du Grand-Livre exporté pour la DGI

Recommandations :

  - Format : PNG (ou JPG si pas de transparence)
  - Ratio  : portrait, environ 290 / 440 (proche d'un A4)
  - Taille : 580 × 880 px ou plus pour un rendu net en rétina
  - Recadrage : la page entière (le rendu utilise object-fit: contain,
    donc l'image rentre proprement sans crop)

Comment générer ces captures :

  1. Connecte-toi sur ApeX, ouvre chaque document, télécharge le PDF
  2. Ouvre le PDF (Adobe Reader, Preview, ou navigateur)
  3. Capture la 1ʳᵉ page (Cmd+Maj+4 macOS, Print Screen Windows,
     ou outil capture intégré)
  4. Optionnel : passe dans un éditeur (GIMP, Photopea) pour
     normaliser les dimensions ; ce n'est pas obligatoire

Tant qu'un fichier est manquant, la carte correspondante affiche
automatiquement le mockup synthétique de secours (déjà en place).
Tu peux donc déposer les images une par une, dans l'ordre que tu
veux, sans casser le rendu.
